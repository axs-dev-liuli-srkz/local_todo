# TODOリストApp マニュアル

## 操作⽅法

1. TodoListに追加したい作業項⽬を  
⼊⼒フォームに⼊⼒して  
Add Todoボタンを押下

2. 終了した作業項⽬の  
Doneボタンを押下すると  
DoneListに移動

3. DoneListの作業項⽬の  
Returnボタンを押下すると  
TodoListに差し戻し

4. Deleteボタンを押下すると  
作業項⽬を削除

5. ページを閉じたあとでも  
同⼀の端末、ブラウザであれば  
データは保存されます。

※データの保存⽅式はWebStorageで端末のブラウザに保存されます。  
⼊⼒したデータはWebに送信されないので  
セキュリティー・プライバシー⾯でも安⼼してご利⽤いただけます。
